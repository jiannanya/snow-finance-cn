# 分析框架方法论

> 七大分析模块的详细方法论，基于A股市场实战经验提炼。

---

## 模块 A：公司基本面分析

### A1 主营业务画像

**需要回答的核心问题：**
1. 公司卖什么？（产品/服务）
2. 卖给谁？（目标客户群体）
3. 怎么挣钱？（盈利模式）
4. 护城河在哪里？（竞争壁垒）

**数据获取：**
- 同花顺经营分析 > 主营介绍 > 董事会经营评述
- 公司官方网站/年报业务介绍章节

**输出格式：**
```
主营业务：[简洁描述，1-2句话]
收入构成：[按产品/地区/客户类型拆分，附占比]
商业模式：[B2B/B2C/平台/制造等]
核心壁垒：[技术壁垒/规模效应/品牌/供应链/客户粘性]
```

### A2 管理层评估

| 维度 | 评估要点 |
|------|---------|
| 控制权 | 实际控制人持股比例，是否有一致行动人 |
| 战略执行 | 历年业绩是否达成业绩目标/承诺 |
| 减持记录 | 高管/大股东减持历史，金额、时机 |
| 激励机制 | 股权激励计划，行权条件合理性 |
| 信披质量 | 公告语言是否清晰，是否有主动沟通行为 |

## 模块 B：财务健康度分析

### B1 成长性分析（增速框架）

```
营收增速分层：
- <10%   → 成熟/停滞期
- 10-30% → 稳定成长期
- 30-100% → 快速成长期（关注是否可持续）
- >100%  → 爆发期（强周期/新产品放量，需研判持续性）
- 负增长  → 衰退期或行业下行，重点关注原因

净利润增速 vs 营收增速：
- 利润增速 >> 营收增速 → 盈利能力提升（毛利率改善/规模效应）
- 利润增速 << 营收增速 → 盈利能力下滑（价格战/成本压力）
- 利润增速 >> 0 但营收负增长 → 注意降本因素是否可持续
```

### B2 盈利质量评估

| 指标 | 优秀标准 | 警示信号 |
|------|---------|---------|
| 净利率 | 行业TOP 25% | 持续低于行业均值 |
| 毛利率 | 稳定或提升 | 连续下滑超3个季度 |
| ROE | >15% | <8% 或持续下滑 |
| 扣非/归母净利比 | >85% | <70%（非经常性损益占比高）|
| 经营现金流 | >净利润×80% | 长期低于净利润（应收账款占用资金）|

### B3 偿债能力

| 指标 | 合理范围 | 风险阈值 |
|------|---------|---------|
| 资产负债率 | 30-65% | >70%（重资产行业除外）|
| 流动比率 | >1.5 | <1.0（短期流动性危机风险）|
| 速动比率 | >1.0 | <0.5 |
| 有息负债率 | <40% | >60%（利息负担重）|

> **行业差异修正**：银行/地产/重工业资产负债率天然高，需与同行业比较而非绝对值判断。

### B4 运营效率

```
存货周转天数 = 365 / 存货周转率
  - 下降趋势 → 产品销售顺畅，库存管理改善
  - 上升趋势 → 库存积压风险（需结合行业景气度判断）

应收账款周转天数
  - 与客户信用政策、行业惯例对比
  - 上升可能意味着应收账款质量恶化

营业周期 = 存货周转天数 + 应收账款周转天数
```

## 模块 C：估值水平分析

详见 [估值模型手册](./valuation-models.md)

**快速估值定位（三步法）：**

```
步骤1：确定PE区间
  当前PE → 查看历史PE百分位（同花顺价值分析页）
  结论：历史高/中/低位

步骤2：与同业比较
  当前PE vs 行业平均PE vs 头部公司PE
  结论：是否溢价/折价，原因是否合理

步骤3：与增速匹配（PEG）
  PEG = PE / 净利润增速（%）
  PEG < 1：增速支撑当前估值（偏低估）
  PEG 1-2：合理区间
  PEG > 2：需要更高增速支撑（偏高估）
```

## 模块 D：行业景气度分析

### D1 行业周期判断框架

```
强周期行业（半导体/钢铁/化工/航运）：
├── 供给端：产能利用率、新增产能周期（3-5年建设周期）
├── 需求端：下游应用场景需求增速
├── 库存端：供应链库存周期（去库/补库）
└── 价格端：产品价格涨跌幅、期货升贴水

景气度判断信号：
- 行业进入上行周期 → 产品价格上涨 + 库存去化 + 开工率提升
- 行业进入下行周期 → 价格下跌 + 库存堆积 + 产能扩张过快
```

### D2 催化剂与政策

| 催化剂类型 | 示例 | 持续性 |
|-----------|------|--------|
| 技术革命 | AI算力需求爆发驱动存储需求 | 中长期（3-5年） |
| 国产替代 | 进口替代政策加速国内供应链 | 中长期（3-5年） |
| 周期反转 | NAND/DRAM供需缺口收窄 | 中期（1-3年） |
| 政策补贴 | 新能源汽车补贴/半导体大基金 | 短中期（1-2年） |
| 突发事件 | 地缘政治断供/自然灾害 | 短期（3-12个月）|

### D3 竞争格局分析（波特五力）

```
1. 同行竞争者：主要竞争对手，市占率分布
2. 新进入者威胁：技术壁垒/资本门槛/牌照许可
3. 替代品威胁：技术颠覆可能性
4. 供应商议价：上游集中度，原材料价格波动
5. 客户议价：客户集中度，切换成本
```

## 模块 E：资金动向分析

### E1 资金流向解读

```
主力（大单）净流入判断：
├── 连续3日净流入 + 股价上涨 → 趋势性建仓，信号强
├── 净流入 + 股价下跌       → 主力逢低买入，短期可能整理
├── 净流出 + 股价上涨       → 主力借势出货，注意风险
└── 持续净流出              → 主力离场，谨慎

融资余额趋势：
├── 融资余额持续增加 → 杠杆做多意愿强，市场看多
├── 融资余额快速下降 → 止损/去杠杆，可能触发踩踏
└── 融资余额/流通市值占比：>5%属于融资盘较重
```

### E2 龙虎榜解读

```
机构专用席位买入：
- 多家机构集体扫货 → 机构一致看多，信号强度高
- 深股通/沪股通买入 → 外资看好，通常偏中长线
- 游资营业部买入    → 短线炒作，注意跟风风险

龙虎榜出现场景：
- 日涨幅偏离值达7%  → 主动监控
- 日换手率达20%     → 异动监控
- 连续3日涨幅偏离   → 持续监控
```

### E3 机构持仓变动

```
季报/年报公布后分析机构持仓：
- 新晋机构数量增加  → 机构发现价值
- 持仓比例提升      → 加仓信号
- 主动基金持仓      → 长线投资，稳定性强
- 量化基金持仓      → 可能跟趋势，波动性大
```

## 模块 F：风险矩阵

### F1 标准风险评级模板

| 风险编号 | 风险类型 | 触发条件 | 影响程度（1-5） | 概率（1-5） | 综合评分 | 应对建议 |
|---------|---------|---------|--------------|-----------|---------|---------|
| R1 | 行业下行周期 | 存储价格下跌>30% | 5 | 3 | 15 | 跟踪月度价格指数 |
| R2 | 存货减值 | 产品价格低于成本价 | 4 | 2 | 8 | 关注存货结构和账期 |
| R3 | 客户流失 | 主要客户份额下降 | 4 | 2 | 8 | 跟踪客户集中度变化 |
| R4 | 大股东减持 | 大宗交易/公告减持 | 3 | 3 | 9 | 关注减持预披露公告 |
| R5 | 监管变化 | 出口管制/政策收紧 | 4 | 2 | 8 | 关注行业政策动态 |

> 综合评分 = 影响程度 × 概率，≥12为高风险，6-11为中等风险，<6为低风险

### F2 A股特有风险点

```
减持风险：
- 解禁日期前后（上市满1年/3年限售股解禁）
- 高管、大股东、创投机构减持预期
- 股权激励行权后的潜在减持

市场风险：
- 北交所/创业板流动性风险（成交量小，买卖价差大）
- 大股东质押股权比例（>50%存在强制平仓风险）
- ST/*ST退市风险

监管风险：
- 财务造假风险（重点关注存货大增+现金流差的组合）
- 关联交易异常
- 审计意见非标准无保留意见
```

## 模块 G：综合研判输出

### G1 投资逻辑梳理（多空双方）

```
看多逻辑（Bull Case）：
□ 行业处于上升周期 / 政策催化
□ 公司竞争地位提升
□ 业绩增速超预期
□ 估值有安全边际（历史低位/PEG<1）
□ 机构加仓 / 主力资金持续流入

看空逻辑（Bear Case）：
□ 周期见顶，价格反转风险
□ 竞争加剧，市场份额下降
□ 财务风险：负债率高/现金流差
□ 估值偏贵（历史高位/PEG>2）
□ 大股东减持压力
```

### G2 综合评分卡

| 维度 | 权重 | 得分(1-5) | 加权得分 |
|------|------|---------|---------|
| 盈利爆发力 | 20% | — | — |
| 行业景气度 | 20% | — | — |
| 估值吸引力 | 15% | — | — |
| 财务健康度 | 15% | — | — |
| 竞争壁垒 | 15% | — | — |
| 资金面 | 10% | — | — |
| 管理层与治理 | 5% | — | — |
| **综合得分** | 100% | — | **—/5** |

```
评分区间：
4.0-5.0分 → 强烈推荐关注（做多）
3.0-4.0分 → 推荐关注
2.0-3.0分 → 中性/观望
1.0-2.0分 → 谨慎/回避
```
