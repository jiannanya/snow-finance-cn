# 估值模型手册

> A股常用估值方法详解，含行业PE参考表与情景分析框架。

---

## 一、估值方法总览

| 方法 | 适用场景 | 核心公式 | 局限性 |
|------|---------|---------|--------|
| **PE法** | 盈利稳定的成长/周期股 | 目标价 = 远期EPS × 目标PE | 亏损股失效 |
| **PB法** | 重资产/金融/地产股 | 目标价 = BVPS × 目标PB | 轻资产股失真 |
| **DCF法** | 现金流稳定的成熟企业 | 所有未来FCF折现之和 | 假设敏感，误差大 |
| **EV/EBITDA** | 跨国/跨行业比较 | EV = 市值+净债务；EBITDA=息税折旧摊销前利润 | 计算复杂 |
| **PS法** | 亏损/早期成长企业 | 目标价 = 预期营收/股本 × 目标PS | 忽略盈利质量 |
| **PEG法** | 快速成长股 | PEG = PE/净利增速；PEG=1合理 | 高增速线性外推风险 |

## 二、PE估值法（最常用）

### 2.1 基本公式

$$\text{目标价} = \text{远期EPS} \times \text{目标PE倍数}$$

$$\text{远期EPS} = \frac{\text{预测净利润（归母）}}{\text{总股本}}$$

### 2.2 远期EPS预测方法

```
方案A：机构研报一致预期
  - 取≥3家机构预测平均值
  - 来源：同花顺盈利预测页、Wind（如有）

方案B：趋势外推
  - 基于历史增速×(1+修正系数)
  - 修正因子：行业景气变化、公司战略变动

方案C：季度数据年化（最新且有季报时优先使用）
  - 年化EPS = 季度EPS × 4
  - 适用：当前季度业绩有确定性爆发时
  - 注意：强季节性行业需做季节性修正

最佳实践：三种方案均计算，取加权平均
```

### 2.3 目标PE倍数选取

**原则一：历史PE百分位法**
```
查同花顺"价值分析"页，获取历史PE分位数：
- 当前PE < 历史25%分位 → 历史低估区域，可用均值上浮
- 当前PE在25-75%区间  → 历史合理区域，用历史均值
- 当前PE > 历史75%分位 → 历史高估区域，目标PE保守给出
```

**原则二：行业PE比较法**

| 行业 | 牛市PE区间 | 正常PE区间 | 熊市PE区间 |
|------|-----------|-----------|-----------|
| 半导体设计 | 40-80x | 25-50x | 15-30x |
| 半导体存储/封测 | 20-40x | 12-25x | 8-15x |

| 新能源（电池）| 25-40x | 15-25x | 10-18x |

**原则三：周期股折价法**
```
对于明显周期行业（半导体/化工/钢铁/航运）：
- 不可在周期高点按高利润给高PE（双击泡沫）
- 应在周期低点时，按归一化利润估值（穿越周期PE）
- 常见做法：用近3年平均利润 × 中位数PE，而非当期峰值利润 × 高PE

德明利案例：
  2026Q1利润爆发期 → 用动态PE 12-15x（存储行业中位数偏低）
  不宜在业绩高点给30x+（存储下行时利润可能腰斩）
```

## 三、PB估值法

### 3.1 基本公式

$$\text{目标价} = \text{每股净资产（BVPS）} \times \text{目标PB倍数}$$

$$\text{BVPS} = \frac{\text{归属母公司股东权益}}{\text{总股本}}$$

### 3.2 目标PB选取参考

| 行业类型 | 正常PB区间 | 说明 |
|---------|-----------|------|
| 银行 | 0.5-1.5x | 资产质量决定PB |
| 保险 | 0.8-2.0x | 内含价值更常用 |
| 房地产 | 0.5-1.5x | 土地储备价值 |
| 制造业（重资产）| 1.5-3.0x | |
| 科技/成长（轻资产）| 3.0-10x+ | 主要靠PE/PS |

## 四、DCF估值法（简化版）

### 4.1 核心逻辑

$$\text{内在价值} = \sum_{t=1}^{n}\frac{FCF_t}{(1+WACC)^t} + \frac{\text{终值}}{(1+WACC)^n}$$

其中：
- $FCF_t$ = 第t年自由现金流（经营现金流 - 资本支出）
- $WACC$ = 加权平均资本成本
- 终值 = $\frac{FCF_{n+1}}{WACC - g}$（g为永续增长率）

### 4.2 A股市场参数参考

```
WACC估算（A股参考值）：
- 无风险利率：中国10年期国债收益率（约2.2-2.8%，2026年参考值）
- 市场风险溢价：5.5-7%（A股历史溢价）
- Beta值：
    * 周期股 β ≈ 1.2-1.5
    * 消费稳定股 β ≈ 0.8-1.0
    * 科技/高成长 β ≈ 1.3-1.6

永续增长率 g：
- 通常取 2-4%（GDP增速以内）
- 注意：g > WACC 会导致终值无限大，不可取

预测期：
- 通常5-10年
- 科技/高成长：10年（需要更长时间兑现价值）
- 成熟行业：5年
```

### 4.3 DCF局限性

```
在以下情形请谨慎使用DCF：
1. 周期行业：FCF预测误差极大
2. 早期成长企业：当前FCF为负
3. 非标准会计处理较多的公司
4. 存在商誉减值/一次性损益较多

建议：DCF作为辅助验证，不作为主要定价依据
```

## 五、情景分析框架

### 5.1 三情景模型

| 情景 | 触发条件 | EPS预测 | 目标PE | 目标价 | 概率权重 |
|------|---------|---------|--------|--------|---------|
| 悲观（熊） | 行业下行/价格回落/业绩不及预期 | 低预测值 | 低区间 | 低目标价 | 20-25% |
| 基础（中） | 行业中性/符合预期 | 中位预测值 | 中位估值 | 合理目标价 | 50-60% |
| 乐观（牛） | 行业超预期/市场情绪高涨 | 高预测值 | 高区间 | 高目标价 | 20-25% |

$$\text{加权目标价} = \sum \text{概率} \times \text{目标价（各情景）}$$

### 5.2 德明利案例填写示例

| 情景 | EPS(元) | PE(x) | 目标价(元) | 概率 |
|------|---------|-------|-----------|------|
| 熊市情景 | 8.00 | 12x | 96.00 | 20% |
| 基础情景 | 10.00 | 15x | 150.00 | 60% |
| 牛市情景 | 12.00 | 20x | 240.00 | 20% |
| **加权目标价** | — | — | **156.00** | 100% |

```
加权目标价 = 96×20% + 150×60% + 240×20%
           = 19.2 + 90 + 48
           = 157.2 元（约157元，对应约+X%上涨空间）
```

## 六、目标价确定流程

```
Step 1: 收集机构一致预期
  → 获取≥3家机构的EPS预测（当年/次年）

Step 2: 自行测算EPS
  → 基于最新季报数据验证机构预测合理性

Step 3: 确定估值区间
  → 结合行业PE参考表 + 历史PE分位 + 周期位置

Step 4: 情景分析三情景
  → 填写熊/基/牛三情景表格

Step 5: 计算加权目标价
  → 加权平均，四舍五入到整数

Step 6: 安全边际检验
  → 目标价相对当前股价的上涨空间（应>15%才有投资意义）
  → 悲观情景目标价是否有足够下跌保护（安全边际）
```

## 七、常见估值误区

```
❌ 误区1：用周期高点利润给高PE（双击泡沫）
   → 正确做法：穿越周期的归一化利润×中位数PE

❌ 误区2：照搬境外公司估值（美股/港股）
   → A股有额外流动性溢价、控制权溢价、散户结构溢价

❌ 误区3：忽略解禁/增发稀释影响
   → 目标价应基于摊薄后EPS（考虑股权激励、可转债稀释）

❌ 误区4：只用单一估值方法
   → 交叉验证：主要方法 + 1-2个辅助方法，结果相互印证

❌ 误区5：忽视流动性折价
   → 小市值股（<30亿市值）需给流动性折价（约15-25%）
   → 北交所股票流动性折价更大（约20-35%）
```
